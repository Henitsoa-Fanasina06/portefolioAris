# Portfolio – Déploiement en ligne

Ce projet est un portfolio statique (HTML/CSS/JS). Vous pouvez le publier facilement pour obtenir un lien public.

## Option 1 – GitHub Pages (recommandé)

1. Créez un dépôt GitHub (ex: `portfolio`).
2. Mettez à la racine du dépôt :
   - `index.html`
   - `styles.css`
   - `script.js`
   - `profile.jpg` (votre photo)
3. Sur GitHub :
   - Allez dans **Settings** → **Pages**
   - **Source** : `Deploy from a branch`
   - **Branch** : `main` et **/ (root)**
   - Enregistrez
4. Votre site sera disponible sur :
   - `https://<votre-utilisateur>.github.io/portfolio/`

## Option 2 – Netlify (rapide)

1. Connectez-vous à https://www.netlify.com/
2. Glissez-déposez le dossier du projet
3. Netlify génère automatiquement un lien public

## Conseils

- Assurez-vous que `profile.jpg` est bien au même niveau que `index.html`.
- Si vous modifiez le site, republiez (ou poussez vers GitHub si vous utilisez Pages).

## Contenu du projet

- `index.html` : structure
- `styles.css` : styles
- `script.js` : interactions
